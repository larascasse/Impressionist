Hey there!

Thanks for checking out impressionist. impressionist is a visual tool that enables you to create impress.js presentations in a visual, easy to use way.Impressionist is conceptualized and developed by Harish Sivaramakrishnan and is under open development on github. Please feel free to watch / fork / log issues.

Why did I build impressionist?

Impress.js expects you to have a reasonably good knowledge of programming with HTML and CSS to make presentations that look good. If you don't do HTML/CSS, impress.js is not for you. Impressionist is an effort to change that. Impressionist has a user interface very similar to powerpoint / keynote which will reduce your learning curve. The orchestration lets you author gorgeous transitions for your presentations visually by leveraging the brilliant 3D/2D transforms that impress.js brings along.

Writing presentations should be fun, and a visual tool can surely help!

Sounds good, can I start using it already?

YES. You can start using impressionist to build your presentations. However, it still has some distance to go before it becomes release quality. Expect to find bugs as you use it :) Impressionist is opensource under MIT license - Do help out fixing issues.

IMPORTANT: impressionist is currently supported on webkit based browsers (Chrome / Safari) only. Running this on Mozilla / Opera / IE will give you erratic results (or it may not work at all ;))

Key features

  • Visual slide authoring
  • Drag and drop positioning of slide elements
  • Supports Text and Images
  • Copy paste support for slide elements
  • Keyboard navigation for slide element repositioning
  • Gorgeous typography with google webfonts
  • Skew, Rotate and transforming of slide elements
  • Visual transition orchestrator
  • Change and preview
  • Download presentation as ZIP
